# User Service

Microservice for user management, authentication, DB access, gRPC, Kafka, Redis, and more.

## Features
- PostgreSQL (Prisma)
- MongoDB (Mongoose)
- Redis caching
- gRPC server
- Kafka producer/consumer
- JWT auth, RBAC (TODO)
